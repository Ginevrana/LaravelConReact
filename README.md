<h3>Prerrequisitos</h3>

<h3>Instalación</h3>
Instalación y configuración de la aplicación

1. Clonar el repositorio 

   ```
   git clone https://github.com/Ginevrana/laravelconreact.git
   ```
2. Instalar los paquetes de NPM 
   ```
   npm install
   ```
3. Instalar composer
   ```
   composer install
   ```
